[Yjs](https://yjs.dev/) port for Elixir using [y-crdt](https://github.com/y-crdt/y-crdt)


A demo using the phoenix framework can be found [here](https://github.com/satoren/y-phoenix-channel).

## Installation

```elixir
def deps do
  [
    {:y_ex, "~> 0.6.0"}
  ]
end
```
